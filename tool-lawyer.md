# Инструмент: lawyer

## Описание

RAG-юрист — отвечает на юридические вопросы, используя базу законов, загруженную в FAISS-индекс.

Для каждого вопроса: ищет релевантные куски законов → формирует промпт с контекстом → генерирует ответ через локальную LLM.

### Команда

```bash
ai-mini-box lawyer [OPTIONS] QUESTION
```

### Аргументы

| Аргумент | Тип | Описание |
|----------|-----|----------|
| `QUESTION` | STRING | Юридический вопрос |

### Опции

| Опция | Описание |
|-------|----------|
| `--top-k N` | Количество законов для контекста (default: 3) |
| `--index PATH` | Путь к FAISS-индексу (default: data/models/law_index) |
| `--model PATH` | Путь к GGUF-файлу LLM |
| `--json` | Вывод в JSON |
| `--interactive` | Режим диалога (chain of questions) |
| `--clear` | Очистить историю диалога |

### Примеры

```bash
ai-mini-box lawyer "Какие права у потребителя при возврате товара?"
# → "Согласно Закону о защите прав потребителей, ст. 25..."

ai-mini-box lawyer --json "Какой срок гарантии?"
# → {"answer": "...", "sources": ["ГК РФ ст. 477", "ЗоЗПП ст. 19"], "time_ms": 5200}

ai-mini-box lawyer --interactive "Что делать, если товар сломался?"
# → Ответ + приглашение ввести следующий вопрос
```

## Промпт для вайбкодинга

```markdown
Создай CLI-инструмент `lawyer` для RAG-ответов на юридические вопросы.

### Требования:
1. Используй Typer для CLI
2. Используй существующий `LawyerService` из `core/services/lawyer_service.py`:
   - Конструктор: `LawyerService(llm, law_repo, embedder, index_path)`
   - Метод `ask(question: str, top_k: int = 3) -> str`
   - Метод `ask_with_sources(question: str, top_k: int) -> tuple[str, list[str]]`
3. Используй `FaissVectorStore` из `infrastructure/vector_store/faiss_store.py` для поиска
4. Lazy-load: FAISS-индекс и LLM загружаются при первом вызове
5. Флаг `--interactive` для пошагового диалога (сохраняет историю)
6. Флаг `--json` возвращает: answer, sources (список), time_ms
7. Вывод источников в читаемом виде: "ст. N закона M"
8. Если индекс пуст — предложить сначала запустить `ingest-laws`
```
