# Commands

| Command | Description |
|---|---|
| `ai-mini-box telegram poll` | Poll Telegram once and save new messages |
| `ai-mini-box telegram daemon` | Run continuous polling loop until interrupted |

## Web UI

After installing the plugin, open the web interface → Plugins → telegram:

- **Poll now** — run a single poll cycle
- **Start daemon** — start background polling (visible in logs)
- **Stop daemon** — stop background polling
- **View Messages** — navigate to saved messages
