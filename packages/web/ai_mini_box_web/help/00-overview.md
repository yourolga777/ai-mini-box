# AI Box — система для автоматизации бизнеса

AI Box — это программа, которая помогает управлять заказами, сообщениями от клиентов, товарами, задачами и контактами в одном окне. Она работает через браузер (как веб-страница).

## Как запустить

На компьютере с Windows:
1. Откройте папку с программой
2. Дважды кликните на `run.bat`
3. Откроется браузер с главной страницей

Всё остальное программа сделает сама: установит нужные компоненты и запустит сервер.

## Как устроен интерфейс

Слева — боковое меню с разделами:
- **Дашборд** — сводка по всем разделам (счётчики, графики, аналитика)
- **Сообщения** — все входящие сообщения
- **Контакты** — клиенты и партнёры
- **Заказы** — оформленные заказы с детальной карточкой
- **Товары** — номенклатура
- **База знаний** — полезная информация для ИИ
- **Календарь** — задачи и записи
- **Плагины** — дополнительные возможности (Telegram, ИИ, Email и др.)
- **Справка** — этот раздел
