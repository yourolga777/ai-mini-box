# Заказы

Раздел для учёта заказов от клиентов.

## Что можно делать

- **Просматривать список** — все заказы с указанием статуса, суммы, контакта, даты
- **Открыть заказ** — клик по строке открывает детальную карточку заказа
- **Создать заказ** — кнопка «Добавить», указать контакт, товары, сумму
- **Создать из сообщения** — в карточке сообщения нажмите «Создать заказ» (ИИ может создать заказ автоматически, если клиент оформил заказ в сообщении)
- **Добавить позиции** — в карточке заказа можно указать товары, количество и цену
- **Менять статус** — «Новый» → «В обработке» → «Выполнен». Любой заказ можно отменить.
- **Фильтровать** — по статусу, по контакту, по тексту заметок
- **Искать** — по тексту заметок (строка поиска)

## Связь с контактами

Каждый заказ привязан к контакту. В карточке контакта видно все его заказы. А в карточке заказа — контакт клиента и сообщение, из которого создан заказ.

## Страница заказа

На странице заказа `/orders/123` отображается:

- **Карточка заказа** — статус, контакт, даты, заметки
- **Позиции заказа** — таблица товаров с количеством, ценой и суммой. Итого внизу.
- **История** — заказ можно удалить (позиции удалятся автоматически).
