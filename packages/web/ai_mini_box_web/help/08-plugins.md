# Плагины

Плагины — это дополнительные модули, которые расширяют возможности AI Box. Например, Telegram-бот, ИИ-функции или Email-почта.

## Вкладки

Раздел «Плагины» разделён на две вкладки:

- **Установленные** — плагины, которые уже установлены на вашем сервере
- **Доступные** — плагины, которые можно установить из каталога

## Как управлять плагинами

Для каждого плагина отображается:
- **Название и описание** — что делает этот плагин
- **Версия** — установленная и последняя доступная
- **Статус** — работает / остановлен / не установлен / есть обновление
- **Кнопки управления** — запустить, остановить, настроить, обновить, удалить

## Какие плагины бывают

- **Telegram-бот** — подключает Telegram для приёма сообщений
- **ИИ (LLM)** — добавляет функции искусственного интеллекта: автоответы клиентам, категоризация сообщений (ЗАКАЗ/ВОПРОС/ЖАЛОБА), автораспределение по папкам, черновики ответов, извлечение данных, эскалация оператору.
- **Email** — подключает почтовый ящик для приёма и отправки писем
- **Демо-плагин** — пример для разработчиков

## Установка и обновление

Во вкладке «Доступные» показаны плагины из каталога. Нажмите «Установить» — программа скачает и подключит его.

Если для установленного плагина вышла новая версия, появится кнопка «Обновить». После обновления перезапустите плагин.

## Настройка плагина

У каждого плагина могут быть свои настройки (например, токен Telegram-бота или параметры почтового ящика). Нажмите «Настроить» — откроется форма с полями конфигурации.
