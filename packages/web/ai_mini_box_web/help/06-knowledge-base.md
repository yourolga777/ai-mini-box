# База знаний

Раздел для хранения полезной информации, которую может использовать искусственный интеллект при ответах клиентам.

## Для чего нужна

Если подключён ИИ-плагин, то при составлении ответа клиенту ИИ будет учитывать информацию из базы знаний. Например, вы можете записать туда:

- **Прайс-лист** — актуальные цены на товары и услуги
- **График работы** — часы работы, выходные дни
- **Условия доставки** — сроки, стоимость, регионы
- **Часто задаваемые вопросы** — ответы на типовые вопросы клиентов
- **Контакты сотрудников** — кто за что отвечает

## Как заполнять

1. Нажмите «Добавить запись»
2. Укажите название (например, «Условия доставки по Москве»)
3. Напишите содержание
4. Сохраните

Каждую запись можно редактировать или удалять.
