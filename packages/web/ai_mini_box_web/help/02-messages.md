# Сообщения

Сюда попадают все входящие сообщения из подключённых каналов: Telegram, email и других.

## Что можно делать

- **Просматривать список** — все сообщения в виде таблицы: текст, отправитель, дата, тема, категория
- **Искать** — строка поиска над таблицей (по тексту сообщения, по отправителю)
- **Сортировать** — клик по заголовку колонки «Дата» или «Тема»
- **Фильтровать по категории** — выпадающий список "Все категории" / "ЗАКАЗ" / "ВОПРОС" / "ЖАЛОБА" и т.д.
- **Открыть сообщение** — клик по строке, чтобы увидеть полный текст
- **Привязать к контакту** — если сообщение от неизвестного, можно привязать к существующему контакту
- **Назначить папку** — распределить сообщение по тематической папке (см. раздел «Папки»)
- **Перетащить в папку** — захватите сообщение мышкой (или пальцем на планшете) и перетащите на нужную папку в боковой панели
- **Массовое назначение** — отметьте несколько сообщений чекбоксами, выберите папку — все отмеченные назначатся за раз
- **Создать заказ из сообщения** — в карточке сообщения нажмите «Создать заказ», если клиент оформил заказ

## Категории сообщений

При подключённом LLM-плагине каждое сообщение анализируется ИИ и получает категорию:

| Категория | Значение | Цвет бейджа |
|---|---|---|
| ЗАКАЗ | Клиент хочет сделать заказ | Синий |
| ВОПРОС | Общий вопрос | Зелёный |
| ЖАЛОБА | Претензия или жалоба | Красный |
| ПРЕДЛОЖЕНИЕ | Идея или предложение | Серый |
| ФЛУД | Не по делу | Оранжевый |

Категория отображается в таблице в виде цветного бейджа. Рядом могут быть значки:

- **🤖** — ИИ уже отправил автоматический ответ на это сообщение
- **👤** — сообщение требует внимания оператора

## Автоматические ответы

ИИ-чатбот (при установленном LLM-плагине и настроенном Telegram-боте) может автоматически отвечать клиентам. В карточке сообщения при этом отображается зелёный блок с текстом автоответа.

Если ИИ определил, что нужен оператор — показывается красный блок "Требует внимания оператора" с контекстом для оператора.

## Переобработать через ИИ

В карточке сообщения есть кнопка **"🔄 Переобработать через ИИ"**. Она повторно отправляет сообщение на анализ в ИИ-чатбот. Это полезно, если вы изменили настройки бизнеса или хотите принудительно обновить категорию/автоответ.

## Откуда берутся сообщения

Сообщения автоматически подтягиваются из подключённых каналов. Например, после настройки Telegram-бота все входящие сообщения из Telegram будут появляться в этом разделе.
