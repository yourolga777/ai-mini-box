# Installation

## Via pip

```bash
pip install ai-mini-box-core ai-mini-box-web
pip install ai-mini-box-demo
```

## From source (development)

```bash
git clone https://github.com/yourolga777/ai-mini-box.git
cd ai-mini-box
pip install -e packages/core/
pip install -e packages/web/
```

## Quick start

```bash
ai-mini-box init
ai-mini-box serve
```

Open http://127.0.0.1:8080 in your browser.

## One-click run (Windows)

Double-click `run.bat` in the project root — it installs dependencies, initializes the project, and starts the server.

Double-click `stop.bat` to stop the server.
