# Товары

Раздел для ведения номенклатуры — списка товаров или услуг.

## Что можно делать

- **Просматривать список** — все товары с названием и ценой
- **Добавить товар** — кнопка «Добавить», указать название, цену
- **Редактировать** — изменить название или цену
- **Удалить** — кнопка удаления
- **Искать** — строка поиска по названию товара

## Цены

Цены указываются в рублях (с копейками). При оформлении заказа можно выбрать товар из списка.
