# Календарь и задачи

Раздел для планирования: задачи, встречи, напоминания.

## Что можно делать

- **Просматривать задачи** — список всех задач с датой, временем, приоритетом
- **Добавить задачу** — указать название, дату, время (опционально), приоритет
- **Отметить выполненной** — переключить статус
- **Редактировать** — изменить дату, время или описание
- **Создание из сообщения** — если в сообщении клиента упоминается дата (например, «встретимся 5 марта»), ИИ может автоматически создать задачу

Задачи можно привязывать к контактам, чтобы видеть все запланированные дела по конкретному клиенту.
